作者：郭康

基于我本人早年辍学，且长期处于低接触、低交流、近乎封闭之现实状态的事实，  除零散观点、故事外，  我不曾系统性学习任何学科体系；

亦长期不处于能够持续吸收、系统整理并重构既有理论体系的现实条件之中。



在此前提下，  原点学并非由多人协作完成；

并非建立在系统学科训练之上；

亦并非产生于高接触、高交流、高输入环境之中。 

其起点、路径与形成方式，  均形成于长期边缘、低接触、近乎封闭的现实条件之下。



也正因如此，  原点学之形成，并非既有理论路径的自然延伸；  

而是在现实之中，  由我本人在长期边缘、低接触、近乎封闭的状态下，  持续推进，并逐步形成连续文本序列。

截至目前，  原点学已不止于一篇单独文字；

其核心概念、问题入口、基本结构与阶段推进，  均已形成。



原点学核心等式为：

原点 = 主体 = 存在 = 可校 = 文明

其中：

原点，  指判断从何开始，问题从何成立，边界从何被划出。

主体，  指谁在承担判断，谁在承担边界，谁在承担代价。

存在，  指其并非空言，  而是在现实之中确实成立、确实发生、确实起作用。

可校，  指其可以被区分，可以被追问，可以被检验，可以被校正。

文明，  指上述结构在更大尺度上的持续展开；  

个体、关系、秩序、边界与代价，  皆须在其中获得稳定归属。

此等式不是修辞，  不是类比，  而是同一问题在不同尺度上的同构表达。

下面只举一个最简单的例子，  说明原点学与已知视角的差别。



学习：

学生、学校、老师、体系、父母，主体不同，  

对“学习”这一问题的原点，必然各不相同；  

原点不同，后续判断、边界、代价与校正，亦必然不同。

若学生主体在学习，  

而其原点不在学习，  

可校亦不在学习，  则“学习”二字于该主体而言，便无法真正成立。

问题不只在于理解不同；  

更在于谁有资格定义学习，  

谁承担学习之代价，  

谁拥有学习之校正权。



若基于学生主体，什么算学习；  

若基于学校主体，什么算学习；  

若基于父母主体，什么算学习。



主体不同，  原点不同；  

原点不同，  存在不同；  

存在不同，  可校不同；  

可校不同，  文明的成立方式亦不同。

上述内容，  仅为原点学问题入口之一，  并非其全部。
