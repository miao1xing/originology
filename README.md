# 原点学（Originology）

原点学，不首先提供答案，而首先区分问题。  

它所关心的，不只是“对”与“错”，  
也不是单纯的立场之争；  
它更关心的是：

- 判断从何开始
- 边界如何成立
- 代价如何归属
- 结构如何稳定
- 个体、群体与文明何以失稳，又何以自保

因此，原点学并不试图直接替代现有学科；  
它更像一套最低结构语言，  
用于重新整理那些常被混淆的问题，  
并划分哪些问题可以解决，哪些问题只能发现。
